# Code EJ

初心者から中級者へ向けて、SwiftUIのUIレイアウトを段階学習できる静的サイトです。

## 技術構成
- HTML
- CSS
- JavaScript

## ページ構成
- `index.html`（ポータル + 旧リンク互換セクション）
- `pages/roadmap.html`
- `pages/text.html`
- `pages/image.html`
- `pages/padding.html`
- `pages/overlay.html`
- `pages/vstack.html`
- `pages/stacks.html`
- `pages/spacer-frame.html`
- `pages/button.html`

## ディレクトリ
- `css/` : `base.css`, `layout.css`, `components.css`, `pages.css`
- `js/` : `main.js`, `code-switcher.js`
- `assets/images/` : 学習補助図版

## 互換性
旧リンク互換のため、`index.html` に以下のアンカーIDを維持しています。
- `#intro`
- `#basics`
- `#sample`
- `#faq`
- `#contact`

## 使い方
`index.html` をブラウザで開いて閲覧してください。
