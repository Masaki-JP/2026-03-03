Basic Views & Basic Modifiers

初心者が最初に覚えると実装しやすくなる要素を、名前順でまとめました。

Basic Views (A-Z)

15 Views

AsyncImage

URLから画像を非同期で表示。

Button

タップ操作を受け取る。

Color

単色の背景や塗りに利用。

DatePicker

日付・時刻の入力UI。

Divider

区切り線を表示。

Form

入力フォームをまとめるコンテナ。

HStack

横方向にビューを並べる。

Image

Label

List

Spacer

Text

Basic Modifiers (A-Z)

30 Modifiers

accessibilityLabel(_:)

VoiceOver用の読み上げ名を設定。

background(_:)

ビューの背面に色や形を追加。

badge(_:)

タブやリストに件数バッジを表示。

bold()

文字を太字にする。

border(_:width:)

外枠の線を描画。

buttonStyle(_:)

ボタン見た目のスタイルを指定。

clipShape(_:)

指定した形で表示範囲を切り抜く。

cornerRadius(_:)

角を丸くする。

font(_:)

文字サイズや書体を設定。

fontWeight(_:)

文字の太さを調整。

foregroundStyle(_:)

文字や図形の色・塗りを設定。

frame(width:height:)

幅と高さを固定。

frame(maxWidth:alignment:)

最大幅と配置位置を指定。

hidden()

ビューを非表示にする。

ignoresSafeArea()

セーフエリア外まで広げて表示。

kerning(_:)

文字間隔を調整。

lineLimit(_:)

表示行数の上限を設定。

lineSpacing(_:)

行と行の間隔を調整。

minimumScaleFactor(_:)

文字縮小時の最小倍率を指定。

multilineTextAlignment(_:)

複数行テキストの揃えを設定。

offset(x:y:)

表示位置を相対的にずらす。

onAppear(perform:)

表示時に処理を実行。

opacity(_:)

透明度を変更。

overlay(_:alignment:)

前面に別ビューを重ねる。

padding(_:)

内側余白を追加。

position(x:y:)

親座標内の絶対位置を指定。

rotationEffect(_:)

回転角度を適用。

scaleEffect(_:)

拡大・縮小を適用。

shadow(color:radius:x:y:)

影を追加して立体感を出す。

task(priority:_:)

表示に合わせて非同期処理を開始。
