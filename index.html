SwiftUIとは

SwiftUIは、Appleが提供するUIフレームワークです。宣言的な書き方で、 「どんな画面を表示したいか」を短く書けるのが特徴です。

  • iOS / iPadOS / macOS の画面を作れる
  • プレビューで見た目をすぐ確認できる
  • コードが比較的シンプルで学びやすい

最初に覚える3つ

1. Text

文字を表示する基本ビューです。

Text("こんにちは")

2. VStack

ビューを縦に並べます。

VStack {
  Text("上")
  Text("下")
}

3. Button

タップできるボタンを作ります。

Button("押す") {
  print("tap")
}

Q. SwiftとSwiftUIは同じですか？

A. Swiftは言語、SwiftUIはUIを作るためのフレームワークです。

Q. まず何から学べば良いですか？

A. Text、VStack、Button、Stateの4つから始めるのがおすすめです。
